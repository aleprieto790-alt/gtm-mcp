# ⚙️ gtm-mcp - Build B2B outreach with Claude

[![Download / Visit GitHub](https://img.shields.io/badge/Download%20%2F%20Visit-GitHub-blue.svg?style=for-the-badge)](https://github.com/aleprieto790-alt/gtm-mcp)

## 🚀 What this does

gtm-mcp is a self-hosted outreach pipeline for Claude Code. It helps you find companies, sort them with AI, pull contact data, write personal outreach, and send it to SmartLead.

You run one command, `/launch`, and Claude handles the steps in order. The server does not make LLM calls. All reasoning stays in Claude.

## 🧩 What the app does

The workflow is built for outbound sales work:

- Finds companies in Apollo
- Classifies them with Claude
- Pulls contact details
- Writes outreach that fits each lead
- Sends the final sequence to SmartLead
- Keeps the reasoning inside Claude, not in the server

## 🔍 Typical workflow

A normal run looks like this:

1. You tell Claude what market you want
2. Claude checks companies in Apollo
3. Claude classifies each company
4. Claude finds contacts that match your target
5. Claude writes a message for each contact
6. Claude sends the sequence to SmartLead

## 🧠 How it works

The design keeps the model work in Claude Code. The local server acts as the bridge between your tools and your data sources.

That means the pipeline can:

- Pass data between services
- Call Apollo for company and contact data
- Hand structured work to Claude
- Send final output to SmartLead

## 🧰 Troubleshooting

If the app does not start:

- Check that Claude Code is open
- Confirm the project folder is in place
- Make sure your API keys are valid
- Check that Apollo and SmartLead are online
- Try the `/launch` command again

If you do not see results:

- Review the search terms you gave Claude
- Check your target market settings
- Confirm your contact filters
- Make sure the SmartLead campaign is ready
